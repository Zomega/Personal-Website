Simmons Hall Dashboard


Dashboard is a HTML5 Application displayed on multiple television screens throughout Simmons Hall at MIT. It's designed to provide residents with information they use on a daily basis. I did most of the front end styling for Dashboard using LESS and CoffeeScript. A development version of the site is here. Dashboard has a very specific deployment target, so little effort has been made to ensure browser compatibility.
